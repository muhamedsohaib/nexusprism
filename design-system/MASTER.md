# Nexus Prism Design System

## Strategic direction

Nexus Prism is a premium UAE AI commerce command center, not a generic AI agency website.

The locked visual formula is:

```txt
Dark luxury palette + restrained liquid glass + editorial spacing + prism light language + dashboard-as-brand
```

## Primary conversion path

```txt
Homepage -> AI Commerce Score -> WhatsApp audit -> paid fix sprint -> monthly retainer
```

## Visual personality

- Luxury-tech first
- Operator-serious second
- Dark cinematic surface
- Warm gold accent
- Cool prism cyan/violet highlights
- Minimal copy with strong hierarchy
- Proof-led sections
- Dashboard previews as brand proof

## Typography

- English display: Cormorant Garamond
- UI/body: Manrope
- Arabic-ready fallback: Noto Sans Arabic

## Anti-patterns

Avoid:

- Generic AI purple gradients
- Loud cyberpunk treatment
- Too many glass panels
- Over-animated page transitions
- Cartoon SaaS visuals
- Crowded agency service-card grids
- Unsupported claims about automation or revenue
